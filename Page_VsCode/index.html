Maîtriser Git dans VS Code

Découvrez comment utiliser Git directement depuis l'interface de VS Code, sans ligne de commande. Suivez ce guide étape par étape pour maîtriser :

  • L'initialisation d'un dépôt (Init)
  • L'ajout de fichiers au suivi (Add)
  • La création de commits
  • La gestion des branches
  • La fusion de code (Merge)
  • La synchronisation avec GitHub (Push/Pull)
